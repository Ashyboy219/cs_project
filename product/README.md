# Frogger — Product

Python + pygame Frogger game for the IB CS SL year-end project.

## Run

```bash
pip install -r requirements.txt
python3 frogger.py
```

Press **Space** to start. Use **arrow keys** or **WASD** to hop.

## Files

| File | Purpose |
|------|---------|
| `frogger.py` | Full game (Frog, Car, Log, Lanes classes + main loop) |
